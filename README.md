
# 🧦 SOCKS5 Browser Configurator

> Удобное расширение для Google Chrome, позволяющее в пару кликов настроить работу браузера через SOCKS5-прокси с гибким списком исключений (bypass list).

## 📖 О проекте

**SOCKS5 Browser Configurator** — это лёгкое расширение для браузера Google Chrome (и других браузеров на движке Chromium), которое позволяет:

- 🔌 Включать и выключать проксирование трафика через SOCKS5-прокси одним переключателем.
- 🌐 Указывать адрес прокси в виде удобных раздельных полей — **Хост** и **Порт**.
- 📋 Настраивать **список исключений (Bypass List)** — домены, которые будут открываться напрямую, минуя прокси.
- 🇷🇺 Работать с национальными доменами (IDN), например `*.рф` или `пример.рф` — они автоматически конвертируются в Punycode.
- 🟢 Видеть текущий статус («Активно» / «Не активно») прямо в интерфейсе, а также по иконке расширения.

Расширение построено на современном **Manifest V3** и не требует внешних зависимостей.

---

## 👥 Кому это пригодится

Расширение будет полезно, если вы:

- 🧑‍💻 **Разработчик**, которому нужно быстро переключать браузер на локальный SOCKS5-прокси (например, `localhost:1080`).
- 🔐 Используете **SSH-туннель** (`ssh -D 1080 ...`) и хотите направлять трафик браузера через него.
- 🌍 Хотите **гибко управлять**, какие сайты идут через прокси, а какие — напрямую.
- ⚙️ Цените **простые и понятные инструменты** без лишних настроек и телеметрии.

## 🚀 Установка

Расширение устанавливается вручную в режиме разработчика (developer mode).

### 1️⃣ Скачайте проект

Клонируйте репозиторий:

```bash
git clone https://github.com/mirninec/socks5-browser-configurator.git
```

Или скачайте ZIP-архив кнопкой **Code → Download ZIP** и распакуйте его.

### 2️⃣ Откройте страницу расширений

В адресной строке Chrome перейдите по адресу:

```
chrome://extensions/
```

### 3️⃣ Включите режим разработчика

В правом верхнем углу активируйте переключатель **«Режим разработчика»** (Developer mode). 🛠️

### 4️⃣ Загрузите расширение

Нажмите **«Загрузить распакованное расширение»** (Load unpacked) и укажите папку с проектом.

✅ Готово! Иконка расширения появится на панели браузера.

## 🧑‍🏫 Как пользоваться

### Настройка прокси

1. 🖱️ Кликните по иконке расширения — откроется страница настроек.
2. 🖊️ В секции **Socks5 Proxy** укажите:
   - **Хост** — адрес прокси-сервера (например, `localhost` или `127.0.0.1`, для IPv6 — `[::1]`).
   - **Порт** — порт прокси-сервера (например, `1080`).
3. 🎚️ Включите переключатель **Socks5 Proxy** — бейдж сменится на яркое **«Активно»**.
4. 💾 Нажмите **«Сохранить»**.

Теперь трафик браузера идёт через ваш SOCKS5-прокси. 🎉

### Список исключений (Bypass List)

Домены из этого списка будут открываться **напрямую**, минуя прокси.

1. 🎚️ Убедитесь, что переключатель **Bypass List** включён.
2. 🖊️ Введите домен в поле **«Новый домен»**. Поддерживаемые форматы:
   - `example.com` — конкретный домен
   - `*.example.com` — домен и все его поддомены
   - `пример.рф`, `*.рф` — национальные домены (IDN)
3. ➕ Нажмите **«Добавить домен»** (или клавишу **Enter**).
4. 🗑️ Ненужные домены удаляются кнопкой **×** рядом с записью.
5. 💾 Не забудьте нажать **«Сохранить»**.

> 💡 **Подсказка:** кнопка «Сохранить» становится активной только при наличии реальных изменений. Если вы вернули настройки в исходное состояние — сохранять нечего.

### Отключение прокси

🎚️ Просто выключите переключатель **Socks5 Proxy** и нажмите **«Сохранить»** — браузер вернётся к системным настройкам сети.

## 🔒 Приватность

Расширение работает **полностью локально**:

- 🚫 Не собирает и не передаёт никакие данные.
- 💾 Все настройки хранятся только в локальном хранилище браузера (`chrome.storage.local`).
- 📡 Не имеет доступа к содержимому посещаемых страниц.

Используются только минимально необходимые разрешения: `storage` (хранение настроек) и `proxy` (управление прокси).

## 🛠️ Технологии

- **Manifest V3** — актуальный стандарт расширений Chrome
- **Service Worker** — фоновая логика и динамическая иконка (OffscreenCanvas)
- **Vanilla JavaScript** — без фреймворков и зависимостей
- **Chrome Proxy API** — управление настройками прокси
