# socks5-browser-configurator
🧦 Расширение для Chrome: настройка браузера через SOCKS5-прокси с гибким списком исключений (bypass list) и поддержкой IDN-доменов. Manifest V3, без зависимостей. / 🧦 Chrome extension to configure your browser with a SOCKS5 proxy, featuring a flexible bypass list and IDN domain support. Manifest V3, zero dependencies.
